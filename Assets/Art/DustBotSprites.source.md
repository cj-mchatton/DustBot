# DustBot generated sprite sources

Mode: OpenAI built-in image generation, followed by chroma-key removal and local sprite preparation.

## Board object atlas prompt

Create a polished casual-mobile-game 4-by-2 sprite atlas on a perfectly flat
`#ff00ff` chroma-key background. The eight isolated top-down objects are:

1. crumpled coral-red sock
2. chunky warm walnut furniture/wall obstacle
3. loose coiled golden electrical cord with a small plug
4. glossy blue water spill
5. colorful toy block
6. cluster of biscuit-brown crumbs
7. mint-and-yellow robot vacuum charging dock
8. cute pale-lavender dust bunny bonus collectible

Use rounded forms, crisp silhouettes, subtle dimensional shading, consistent
deep-teal outlines, generous padding, no text, no watermark, no cast shadows,
and no magenta inside the objects. Every object must remain readable at 64 px.

## Player prompt

Create one charming compact robot vacuum viewed directly from above on a
perfectly flat `#ff00ff` chroma-key background. Give it a circular mint-green
body, cream trim, coral power button, dark-teal face display, friendly bright
eyes, a tiny smile, and subtle side brushes. Keep the silhouette symmetric so
it rotates cleanly, with crisp premium mobile-game rendering, no text, no
watermark, no cast shadow, and no magenta inside the subject.

Generated source images remain in the Codex generated-images directory. The
transparent, trimmed production sprites are stored in
`Assets/Resources/Sprites`.
